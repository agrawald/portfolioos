Portfolio
=============
Portfolio is a maven web application created using following technologies.

Frontend technologies used...
--------------
1. Backbone.js
2. Underscore.js
3. jqueryUI

Backend technologies used...
--------------
1. Spring MVC
2. Spring Mongo

Database used...
--------------
MongoDB

Server Used...
------------
Tomcat jbossews 1.0

Web Hosting...
------------
OpenShift
